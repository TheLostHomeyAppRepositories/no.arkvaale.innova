# Innova Fancoil

App for Innova WiFi connected fancoils

Homey and Fancoil needs to be connected to the same network. App discovers Fancoil through MAC-address.

Atleast know to work with the following models:
- EFA649
- EFB649
- EWF644
- EWF647
